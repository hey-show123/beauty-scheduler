# Beauty Scheduler Frontend 🌟

美容室スケジューリングシステムのReactフロントエンドアプリケーションです。

## 🚀 機能

### 📊 ダッシュボード
- システム統計の表示
- クイックアクションボタン
- システム情報概要

### 👥 スタッフ管理
- スタッフの登録・編集・削除
- スキルレベル管理
- 勤務可能時間設定
- 時給・勤務時間制限設定

### 📅 予約管理
- 顧客情報管理
- サービス選択・組み合わせ
- 優先度設定
- 希望スタッフ指定

### 🎯 スケジュール最適化
- ステップバイステップの最適化設定
- リアルタイム設定プレビュー
- 最適化結果の可視化
- 詳細統計情報

### 📋 スケジュール表示
- タイムライン・グリッドビュー
- スタッフ別フィルタリング
- CSV出力機能
- 印刷対応

## 🛠 技術スタック

- **React** 18.2.0 - UIライブラリ
- **TypeScript** - 型安全性
- **Material-UI** (MUI) - UIコンポーネント
- **React Hook Form** - フォーム管理
- **Axios** - HTTP通信
- **React Router** - ルーティング
- **Date-fns** - 日付操作

## 📦 インストール・実行

### 前提条件
- Node.js 16.x以上
- npm または yarn
- バックエンドAPIが起動していること（http://localhost:8000）

### 1. 依存関係のインストール
```bash
cd frontend
npm install
```

### 2. 開発サーバー起動
```bash
npm start
```

アプリケーションが http://localhost:3000 で起動します。

### 3. ビルド（本番用）
```bash
npm run build
```

## 🔧 API設定

フロントエンドは `package.json` の `proxy` 設定により、開発時に自動的にバックエンドAPI（http://localhost:8000）にプロキシされます。

本番環境では、環境変数またはビルド設定でAPIエンドポイントを設定してください。

## 🔒 型安全性

TypeScriptを使用して以下の型を定義：
- `Staff` - スタッフ情報
- `Booking` - 予約情報
- `Service` - サービス詳細
- `OptimizationResult` - 最適化結果
- APIレスポンス型

## 🎯 主要機能の使い方

### 1. スタッフ登録
1. スタッフ管理ページへ移動
2. 「スタッフ追加」ボタンをクリック
3. 基本情報・スキル・勤務可能時間を入力
4. 「登録」ボタンで保存

### 2. 予約作成
1. 予約管理ページへ移動
2. 「予約追加」ボタンをクリック
3. 顧客情報・サービス・予約日時を入力
4. 希望スタッフや優先度を設定
5. 「登録」ボタンで保存

### 3. スケジュール最適化
1. スケジュール最適化ページへ移動
2. 対象日を選択
3. 対象スタッフと予約を選択
4. 「最適化実行」ボタンをクリック
5. 結果を確認し、「詳細スケジュールを確認」で詳細表示

## 🐛 トラブルシューティング

### よくある問題

**1. APIエラーが発生する**
- バックエンドサーバーが起動していることを確認
- ネットワーク接続を確認
- ブラウザのコンソールでエラー詳細を確認

**2. 画面が正しく表示されない**
- ブラウザのキャッシュをクリア
- 依存関係を再インストール: `rm -rf node_modules && npm install`

**3. ビルドエラー**
- TypeScriptエラーを確認・修正
- 型定義ファイルを確認
